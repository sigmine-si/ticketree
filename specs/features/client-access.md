# 고객 포털 접속

버전: v1.0 · 마지막 변경 2026-07-22

## 이렇게 동작해요

- [ ] (예정 · REQ-002) 관리자가 고객 계정별로 초대 링크와 PIN 번호를 발급할 수 있다
- [ ] (예정 · REQ-002) 발급된 초대 링크와 PIN은 관리자 화면에 한 번만 표시되며, 이후 담당자가 고객에게 직접 전달한다
- [ ] (예정 · REQ-002) 고객은 전달받은 초대 링크에 접속해 PIN을 입력하면 자신의 프로젝트 화면에 들어간다
- [ ] (예정 · REQ-002) PIN을 여러 번 잘못 입력하면 일정 횟수 이후 더 이상 시도할 수 없다
- [ ] (예정 · REQ-002) 초대 링크·PIN을 다시 발급하면 이전 것으로는 더 이상 들어갈 수 없다
- [ ] (예정 · REQ-002) 로그인 없이 아무나 들어갈 수 있던 임시 접속 화면은 이번 배포로 없어진다

## 알려진 제약

- 초대 링크에는 만료 기한이 없다. 재발급하기 전까지는 계속 유효하다
